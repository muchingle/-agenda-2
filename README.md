
# -agenda-2# ベトナムのブルーカラー労働者に関する包括的調査レポート

## TL;DR（要点）
- **規模と賃金**：ベトナムの就業者数は2024年で約5,190万人。うち工業・建設業が約1,680万人、農林水産業が約1,390万人とブルーカラー層が大宗を占める。全国平均月収は770万VND（約304米ドル）にとどまり、農林水産業は490万VND（約187米ドル）と最も低い。
- **海外出稼ぎ**：2024年に15万8,500人超が契約に基づき出国（年間目標の126.9%達成）。渡航先トップ3は**日本・台湾・韓国**で、約50万人の海外労働者が年間約40億米ドルを送金している。各国が選ばれる理由は、日本＝求人が多く参入障壁が低い、台湾＝採用が速い、韓国＝賃金が最も高い（月$1,500～2,000）点にある。
- **言語の壁の克服**：出国前の語学研修（日本語3～6か月、韓国はEPS-TOPIK必須）、政府の言語要件（2027年からの日本「育成就労」でJLPT N4標準化）、そしてNICTの無料音声翻訳アプリ「VoiceTra」（33言語）やJP-MIRAIの23言語相談など、研修・制度・テクノロジーの三層で対応が進む。

---

## エグゼクティブサマリー
本レポートは、ベトナムのブルーカラー労働者の規模・賃金、海外出稼ぎ（労働輸出）の実態、主要な渡航先国とその理由、そして言語の壁を乗り越える方法について、ILO、世界銀行、ベトナム統計総局（GSO）、ベトナム労働傷病兵社会省（MOLISA、2025年以降は海外労働輸出の所管が内務省へ移管）等の権威ある情報源に基づいて整理したものである。

---

## Key Findings（主要な発見）
1. ベトナムの労働市場は依然として「人口ボーナス期」にあり、就業者5,190万人のうち約7割が無資格・低技能のブルーカラー層である。
2. 賃金は年8～9%で上昇しているが、全国平均月収は約304米ドルにとどまり、ベトナム労働総連盟の調査では基本的生活費を賄える労働者はわずか24.5%にすぎない。
3. この国内低賃金が、国内の1.6～5倍の収入を得られる海外出稼ぎの強力な誘因となっている。
4. 渡航先は日本（在留ベトナム人63万人超で国籍別2位）・台湾・韓国に集中し、韓国が単価最高、日本が量・アクセス最良という棲み分けがある。
5. 言語の壁は、出国前研修・政府の語学要件・AI翻訳アプリの併用によって緩和されつつあるが、依然として競争力上の主要課題である。

## Details（詳細）

### 1. ベトナムのブルーカラー労働者数

#### 1.1 労働力全体の規模
GSO（ベトナム統計総局／国家統計局）によれば、2024年のベトナムの15歳以上の労働力人口は約5,300万人（前年比+57.54万人）、就業者数は5,190万人（前年比+58.51万人、+1.1%）であった。労働参加率は68.9%。世界銀行のデータでは2024年の総労働力は約5,713万人とされる。

#### 1.2 産業別の就業構造（2024年）
GSO労働力調査（LFS 2024）に基づく産業3部門別の就業者数は以下の通り。

| 産業部門 | 構成比 | 就業者数（概算） |
|---|---|---|
| サービス業 | 38.7% | 約1,940万人 |
| 工業・建設業 | 33.5% | 約1,680万人 |
| 農林水産業 | 27.8% | 約1,390万人 |

製造業（加工・製造業）は農業に次ぐ第2の雇用部門で、2023年時点で約1,190万人を雇用。農林水産業就業者は構造転換により減少傾向にあるが、依然として最大級の雇用部門である。

#### 1.3 ブルーカラー層の特徴
- 労働力の70%超が「訓練を受けていない（無資格）」労働者であり、学位・資格を持つ労働者は2024年で28.3%にとどまる。
- 就業者の3分の2近くが農村部に居住（2023年で約62.7%が農村部）。
- インフォーマル（非正規・不安定）雇用が就業人口の5分の3超を占める。

### 2. ベトナムのブルーカラー労働者の平均所得

#### 2.1 全国平均所得
GSO発表（2025年1月6日記者会見）によれば、2024年のベトナム人労働者の平均月収は770万VND（約304～305米ドル）で、前年比+8.6%（+61万VND）であった。四半期別では2024年第4四半期に820万VND（約323米ドル）まで上昇。2025年通年では840万VND（前年比+8.9%）となった。

| 区分 | 月収（VND） | 米ドル概算 |
|---|---|---|
| 全国平均（2024年） | 770万 | $304 |
| 男性 | 870万 | $343 |
| 女性 | 650万 | $256 |
| 都市部 | 930万 | $367 |
| 農村部 | 670万 | $264 |

#### 2.2 産業別の賃金（ブルーカラー関連）
- 工業・建設業（製造業を含む）：平均月収 約910万VND（約347.5米ドル）。
- 製造業の工場労働者：2024年で月770万～840万VND（$294～321）。電子・ハイテク製造が高め、繊維・縫製が低め。
- 農林水産業：月収約490万VND（約187米ドル）と最も低い。鉱業は月1,030万VND前後と高い。
- 外資系企業（FIE）は地場企業より賃金が約10～20%高い傾向。

#### 2.3 ブルーカラーとホワイトカラーの比較
- 地域別最低賃金（2024年7月1日改定、+6%）：第1地域496万VND（約$196）～第4地域345万VND（約$137）。2026年1月よりさらに約7.2%引き上げ（第1地域531万VND）。
- ベトナムには明確なホワイトカラーとブルーカラーの賃金格差があり、IT・金融が最も高く、製造・サービスが最も低い。
- ベトナム労働総連盟（VGCL）の2023年調査（労働者3,000人超）では、基本的な生活費を賄える収入があると答えたのはわずか24.5%。76%が残業を厭わず、17%超が頻繁に借金をしていた。

### 3. ベトナム人労働者の海外出稼ぎ（労働輸出）の事例

#### 3.1 労働輸出プログラム（xuất khẩu lao động）の概要
ベトナム政府は「契約に基づくベトナム人海外労働者法（法律69/2020/QH14）」のもと、雇用創出・技能向上・貧困削減・外貨獲得を目的に労働輸出を推進している。海外労働管理局（DOLAB／現・内務省所管の海外労働管理局）が許認可・監督を担う。

- 2021～2024年の4年間で約50万人が契約に基づき出国。
- 現在、約70万人（出典により最大約86万人）のベトナム人労働者が40の国・地域、30以上の業種で就労。
- 年間の送金額は約35億～40億米ドル（海外労働者からの送金）。

#### 3.2 海外で就く職種
製造業、建設業、農業、サービス業（介護、看護、家事労働）が中心。海外労働は国内の同等職種より賃金が大幅に高く、ベトナム全国平均の160～500%に達する。

#### 3.3 具体的な労働者の事例（実名・出典付き）
- **Ngo Van Hien氏（ドンタップ省出身）**：2012年にE-9ビザで韓国の機械工場へ。勤務成績が評価され契約を複数回更新し、2022年にE-7（熟練技能者）ビザを取得。2023年に京畿道金浦市で56㎡のアパートを約28億VNDで購入（うち約20億VNDを年利約4%で銀行借入）。「自分の家を持つと生活が安定する。両親も安心して孫に会いに来られる」（VnExpress International、Le Tuyet記者）。
- **Le Van Lam氏（タインホア省マウラム村出身）**：EPSプログラムで韓国へ。機械加工（金型製作・自動車ドア取付）で月収約8,000万～1億VNDという高収入を5年間得て、帰国後に鹿（鹿茸用）牧場を開業、年約2億VNDの収益（Dân Trí紙、2025年12月、COLAB主催の帰国起業家コンテスト関連報道）。
- **Vu Dinh Gio氏（ラオカイ省バクハ社、元貧困世帯）**：貧困地域向け支援策で韓国へ渡り、温帯果樹栽培で初任給月$1,000。4年間で約10億VNDを貯蓄して帰国し、韓国品種イチゴ等の農場を経営、年2～3億VNDの利益（Dân Trí紙）。
- **Nguyen Van Sac氏（ハノイ出身、38歳）**：2009年に技能実習生として訪日（渡航費約$3,000＋保証金$6,000）。後にエンジニアビザを取得し工場長に昇進。2021年に大阪市生野区で3階建て住宅（約50㎡）を1,600万円（提示18百万円から値下げ、現金払い）で購入。「技能実習を始めた17年前には、日本で家を買えるとは想像もしなかった」（VnExpress International）。
- **Hoang Van Lap氏（タインホア省、タイ族）**：IM Japan（非営利プログラム）で訪日し建設・防水工事に従事、月32～35百万VND（残業除く）を得て月20百万VND超を貯蓄。約7億VNDを持ち帰り、うち5億VNDで両親の家を建設（Dân Trí紙）。

一方、技能実習生の渡航前借金は大きな課題である。出入国在留管理庁の『技能実習生の支払い費用に関する実態調査』（2022年7月26日公表、6か国2,184名対象）では、ベトナム人の来日前借金総額の平均は67万4,480円と国籍別最高（全国籍平均54万7,788円、カンボジア56万6,889円、中国52万8,847円）。来日前に借金している割合もベトナム80.0%・カンボジア83.5%と高い。さらに、来日前に送出機関へ支払った費用の平均は全体52万1,065円のところ、ベトナムは65万6,014円と国籍別最高であった（最低はフィリピンの約9万4千円）。

### 4. ベトナム人移民労働者の主要渡航先トップ3とその理由

#### 4.1 2024年の渡航先別ランキング
2024年通年で15万8,500人超（うち女性5.28万人超）が海外へ出国し、年間目標の126.9%を達成した。MOLISAの2024年1～10月データ（13万640人）に基づく上位国は以下の通り。

| 順位 | 渡航先 | 労働者数（2024年1～10月） |
|---|---|---|
| 1 | 日本 | 62,722人 |
| 2 | 台湾 | 48,533人 |
| 3 | 韓国 | 10,877人 |
| - | 中国 | 1,920人 |
| - | シンガポール | 1,774人 |

2023年は記録的な約16万人が出国し、日本が8万人超、台湾が58,620人、韓国が11,626人を受け入れた。

#### 4.2 各国が選ばれる理由

**日本（第1位）**
- 受け入れ需要が安定的で求人数が多く、言語要件などの参入障壁が比較的低い（技能実習）。
- 訪日資金の借入が容易（ビザ確定後にベトナム国家銀行から借入可能）。
- 在日ベトナム人は63万4,361人（2024年末時点、中国に次ぎ国籍別2位）で日本最大級の外国人コミュニティ。出入国在留管理庁『令和6年末現在における在留外国人数について』によれば「ベトナムは前年からの増加人数が6万9,335人で最多」。在留資格別では技能実習21万2,141人・特定技能13万3,478人・技術・人文知識・国際業務10万8,334人。日本は2028年までに16分野で約82万人の外国人労働者受け入れを見込む。
- 課題：円安により若年層にとっての魅力が低下。2022年2月まで1円＝200ドン超だったが2024年には160ドン割れし、賃金の実質価値が10～20%減少。

**台湾（第2位）**
- 求人要件が比較的緩く採用プロセスが速い。
- 介護・看護、建設、近海漁業、農業、製造が中心。
- 2030年までに40万～48万人の労働力不足が見込まれる。
- 課題：ブローカー手数料が高額（5,000～7,000米ドル）。

**韓国（第3位）**
- 雇用許可制（EPS、E-9ビザ）により高賃金・法的保護が手厚い。MOLISA系の情報（Báo Nhân Dân、COLAB＝海外労働センター）によれば、残業込みの月収は**$1,500～2,000**と海外市場で最高水準。韓国の最低賃金（JILPT資料）は2024年が時給9,860ウォン（前年比+2.5%）、2025年は時給1万30ウォン（前年比+1.7%、初の1万ウォン超え、月209時間換算で月額約206万ウォン）。
- ブローカーを排除した政府間（HRD Korea）の透明な仕組み。
- 課題：EPS-TOPIK（韓国語試験）合格が必須で、契約満了後の不法残留率の高さが一部地域の募集停止につながっている。

#### 4.3 送金と経済効果
2024年には約50万人の海外労働者が約40億米ドルを送金。ベトナム全体の送金額（約160～180億米ドル）の重要部分を占める。

### 5. 言語の壁を乗り越える方法

#### 5.1 ベトナム国内での出国前語学研修
- **日本（技能実習・特定技能）**：合格後、ベトナム国内で3～6か月の日本語研修を受けるのが一般的。登録から出国までの総研修期間は約7.5～12か月。送り出し機関が日本語・職業技能・生活マナーを訓練する。
- **韓国（EPS）**：EPS-TOPIK（雇用許可制韓国語能力試験）の合格が必須。200点満点で80点以上が必要。HRD Korea（韓国産業人力公団）が実施し、製造・建設・農業・漁業の現場用語に重点。出国前に3～5日のオリエンテーション（安全・労働法・文化）も行われる。
- **台湾**：出国前オリエンテーションと基礎的な中国語（北京語）研修。ILOとDOLABが2016年に低技能労働者向けの参加型出国前研修カリキュラムを30の主要送り出し機関に導入した。

#### 5.2 政府が義務付ける言語要件
- 日本：2027年4月開始の新「育成就労」制度では、就労3年でJLPT N4相当の日本語能力が標準・必須化される。特定技能1号はJLPT N4／JFT-Basic、2号はN3相当が必要。技能実習では入国後1か月以内に45時間以上の講習（日本語・法令・安全）が義務。
- 韓国：EPS-TOPIK合格が応募の前提条件。
- 法律69/2020/QH14に基づき、送り出し機関は出国前研修で強制労働・人身取引・ジェンダー暴力の防止に関する知識・技能の提供が義務付けられている。

#### 5.3 現場での言語ツール・翻訳テクノロジー
- **VoiceTra**：日本の情報通信研究機構（NICT）が開発した無料音声翻訳アプリ。NICT公式サイトは「VoiceTra translates between 33 languages」と記載し、ver.9.2.0（2026年2月24日リリース）でタミル語・ベンガル語を追加して33言語対応となった（ベトナム語を含む）。累計約470万ダウンロード。
- **DeepL、Google翻訳、Papago**：文章翻訳・会話翻訳に広く利用。
- **VoicePing**：東京拠点のスタートアップによるAIリアルタイム音声翻訳（日本語・ベトナム語・中国語等に対応、業務用）。

#### 5.4 AI・アプリベースの解決策と支援の実例
- **JP-MIRAI（責任ある外国人労働者受入れプラットフォーム）**：ポータルサイトとアプリで23言語の生活・労働情報を提供。「JP-MIRAI Assist」は週5日（火～土）10:00～18:00に23言語で多言語相談に対応し、東京弁護士会のADRを通じた救済も提供。大林組などの大企業が導入。
- **JITCO日本語学習支援の実例**：ある食鳥処理企業では21人のベトナム人技能実習生がおり、社内でベトナム語ばかり使う傾向があったため、県主催のセミナーを機に2023年8月から日本語勉強会を開始した。
- ベトナム人労働者の自由記述を分析した医学論文（JMA Journal）では、「職場でのコミュニケーション」「文化的差異の克服」が肯定的経験のカテゴリーとして抽出されている。

## Recommendations（提言）
1. **賃金格差の活用と是正**：海外労働は国内賃金の1.6～5倍と高く、貧困削減に有効。ただし高額なブローカー手数料・借金（ベトナム人技能実習生の渡航前借金は平均67万4,480円で国籍別最高）が労働者を脆弱にするため、ILOが推奨するデジタル契約管理・手数料の透明化・直接採用ルートの拡大を優先すべき。判断基準：渡航前借金が最低賃金26か月分（日本の例）を超える場合は要警戒。
2. **言語研修の早期・標準化**：日本の2027年新制度（N4必須化）に備え、出国前研修期間を確保し、JLPT・EPS-TOPIK対策を強化。合格率と研修修了率が目標を下回る場合は、研修期間の延長と公的補助の拡充を検討する。
3. **翻訳テクノロジーの併用**：VoiceTra等の無料公的アプリとJP-MIRAI Assistの多言語相談を現場で併用し、緊急時・行政手続きの障壁を低減。企業は日本語学習会の制度化を併せて行うことが望ましい。
4. **円安モニタリング**：1円＝160ドン割れの状況が続き手取りが目標水準（手取り・家賃控除後で月12万円程度）を下回る場合、韓国・台湾・欧州（ドイツ・ルーマニア等）の高所得市場への分散を検討。

## Caveats（留意事項）
- GSOの「平均月収（770万VND）」は全産業・全雇用形態を含む平均であり、ブルーカラー単独の数値ではない。民間調査（ERI等）の「ブルーカラー平均年収7,650万VND」は調査手法が異なるため単純比較はできない。
- 海外労働者数はMOLISA（企業報告ベース）の数値であり、非正規・不法滞在を含まない。総在外労働者数は出典により60万～86万人と幅がある。
- 2025年に行政再編があり、海外労働輸出の所管がMOLISAから内務省（海外労働管理局）へ移管された点に留意。
- 一部の所得・貯蓄額（月25～35百万VND等）は送り出し機関のマーケティング資料に基づくため、政府統計より信頼性が低い。
- 韓国EPS労働者の月収「$1,500～2,000」はMOLISA／COLABの集計値であり、ILOによる独立した賃金統計ではない点に留意。
